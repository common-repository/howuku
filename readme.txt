## What is Heatmap?

A website heatmap is a visual representation that shows you what elements on your website are getting the most interaction (hot) and those that are getting the least (cold).

You get a comprehensive understanding of what is holding visitorsâ€™ attention, what needs to be improved, and what needs to be removed.

Basically, a heatmap help you see an aggregated view of your user behaviors' hotspot.

## How can heatmap helps you improve website conversions?

1. Make sure all your important Call-to-actions is above the average-fold of your scroll map
2. Identify if your visitors are confused by non-clickable elements and turn those into links
3. Find out what your visitors are looking for and reposition your element to improve engagement rate
4. Make sure visitors seeing critical website elements like eBooks, guides, and solution briefs?

== Installation ==
1. Install Howuku plugin in WordPress and activate it.
2. Create a [new account](https://app.howuku.com/sign-up/basic?utm_source=wordpress) or use existing one to login to Howuku.
3. Find your Howuku code snippet inside Howuku dashboard
4. Paste it into your Howuku plugin on Wordpress and save it
5. Now Howuku is already installed on your website. You can go to Howuku dashboard to view all your website data from there.
